📂 Thư mục Images
==================

Đặt các file hình ảnh của bạn vào đây!

✅ Format khuyến nghị: PNG với nền trong suốt
✅ Kích thước: 128x128px hoặc 256x256px
✅ Tên file: heart1.png, heart2.png, rose.png, v.v.

Ví dụ:
- heart1.png
- heart2.png
- star.png
- flower.png

Sau khi thêm hình, nhớ cập nhật đường dẫn trong file script.js (dòng 40)

const imageUrls = [
    'images/heart1.png',
    'images/heart2.png',
];
